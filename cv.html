Web Developer for Instituto Pedro Nunes Aug 2012 - In Progress

Web developer for Instituto Pedro Nunes (IPN). Currently working on the design and implementation of a system that serves as basis for other projects and allows the dynamic adjustment and extension of the data structure, business rules and presentation of applications even at runtime. The system presents an interface generator (for both RESTful APIs and web components) embedded in a dynamic environment that reacts according to changes in the settings in the way it stores and delivers data.

It also employs a business rules management system to dynamically adjust server-side behavior and gives users an administration area where they can generate thick client-side web applications.
